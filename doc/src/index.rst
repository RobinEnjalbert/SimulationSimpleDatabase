AdaptiveDB
==========

SQL made easy
-------------

The **AdaptiveDB** project provides Python3 tools allowing users to easily develop their **data storage strategies**.

**AdaptiveDB** project was mostly designed to fit the :DeepPhysX:`DeepPhysX <>` and :SOFA:`SOFA <>` frameworks
and is mainly using the :Peewee:`Peewee <>` Python packages.


Features
--------

**AdaptiveDB** project provides the following features:

 * Automatic managing of Database file;
 * Creation of highly customizable Tables in the Database;
 * Easy writing and reading interface;
 * Signal management system;
 * Export Database in JSON and CVS format.
